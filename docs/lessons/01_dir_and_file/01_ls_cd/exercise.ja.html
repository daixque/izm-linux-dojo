📋 課題内容

説明で学んだ「ディレクトリの階層」を、ターミナルと右側のファイルエクスプローラーで確かめましょう。user フォルダから documents サブフォルダへ移動し、中の readme.txt を ls で見つけます。

✏️ やること

  • pwd で現在地を確認します（/home/user — ホームディレクトリのはず）
  • ls で一覧を表示し、documents（ディレクトリ）と notes.txt（ファイル）を見分けます
  • cd documents で documents ディレクトリに移動します（Finder でフォルダを開くのと同じ操作）
  • もう一度 ls を実行し、readme.txt が表示されることを確認します
  • 右のファイルツリーがターミナルの移動と連動していることも確認してみましょう
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • コマンド入力後は Enter キーを押します
  • cd の後にスペースを入れてからディレクトリ名を入力します
  • うまくいかないときは リセット ボタンで最初からやり直せます
