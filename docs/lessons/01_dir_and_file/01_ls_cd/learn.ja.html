ls と cd の基本

📚 説明
English

ターミナルとは？

Linux では、キーボードからコマンドを入力してコンピュータを操作します。その入出力を行う画面をターミナル（端末）と呼びます。

コマンドを入力したら Enter キーを押すと実行されます。

ディレクトリとファイル

Linux のファイルシステムは、フォルダ（ディレクトリ）とファイルの階層構造になっています。

graph TD
    home["/home/user"]
    home --> documents["documents/"]
    home --> notes["notes.txt"]
    documents --> readme["readme.txt"]

基本コマンド

pwd — 現在地を表示

今いるディレクトリのパス（住所）を表示します。

$ pwd
/home/user

ls — ファイル一覧を表示

カレントディレクトリの中身を一覧表示します。

$ ls
documents
notes.txt

cd — ディレクトリ移動

別のディレクトリに移動します。

  • cd documents — documents フォルダに移動（相対パス）
  • cd .. — 一つ上のディレクトリに移動
  • cd /home/user — 絶対パスで移動
