📋 Task

Apply what you learned about directory hierarchy using the terminal and the file explorer on the right. Move from the user folder into the documents subfolder and find readme.txt with ls.

✏️ Instructions

  • Run pwd to check your location (should be /home/user, your home directory)
  • Run ls and tell directories from files — documents vs notes.txt
  • Run cd documents to enter documents (like opening a folder in Finder)
  • Run ls again and confirm readme.txt appears
  • Notice how the file tree on the right updates as you move in the terminal
  • When ready, click the Test button

▶ 💡 Hints

  • Press Enter after typing a command
  • Put a space after cd before the directory name
  • Use the Reset button to start over if needed
