What is a Terminal?

On Linux, you control the computer by typing commands with the keyboard. The screen where you type and see output is called a terminal.

Press Enter to run a command after typing it.

Directories and Files

The Linux filesystem is organized as a hierarchy of folders (directories) and files.

graph TD
    home["/home/user"]
    home --> documents["documents/"]
    home --> notes["notes.txt"]
    documents --> readme["readme.txt"]

Basic Commands

pwd — Print Working Directory

Shows the path of your current directory.

$ pwd
/home/user

ls — List Files

Lists the contents of the current directory.

$ ls
documents
notes.txt

cd — Change Directory

Move to another directory.

  • cd documents — move into documents (relative path)
  • cd .. — move up one directory
  • cd /home/user — move using absolute path
