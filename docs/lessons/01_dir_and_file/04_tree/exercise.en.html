📋 Task

Use tree to inspect the directory layout, then navigate to the deep path projects/web/app.

✏️ Instructions

  • Run tree to see the layout under your home directory
  • Run tree projects to focus on the projects subtree
  • Run cd projects/web/app to reach the app directory three levels down
  • Run pwd and confirm you are at /home/user/projects/web/app
  • Run ls and confirm config.txt is listed
  • When ready, click the Test button

▶ 💡 Hints

  • tree with no arguments lists everything under the current directory
  • Join path segments with /, like projects/web/app
  • Press Tab to complete directory names
