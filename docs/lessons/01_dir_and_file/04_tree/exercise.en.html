📋 Task

Navigate to the deep directory projects/web/app and confirm your location with pwd.

✏️ Instructions

  • Run pwd to check your starting location (/home/user)
  • Run cd projects/web/app to reach the app directory three levels down
  • Run pwd and confirm you are at /home/user/projects/web/app
  • Run ls and confirm config.txt is listed
  • When ready, click the Test button

▶ 💡 Hints

  • Join path segments with /, like projects/web/app
  • You can also go step by step: cd projects, then cd web, then cd app
  • Press Tab to complete directory names
