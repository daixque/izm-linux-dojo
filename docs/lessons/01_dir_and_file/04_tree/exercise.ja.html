📋 課題内容

深い階層にある projects/web/app ディレクトリまで移動し、pwd で正しい場所に着いたか確認しましょう。

✏️ やること

  • pwd で現在地（/home/user）を確認します
  • cd projects/web/app で 3 階層下の app ディレクトリへ移動します
  • pwd で /home/user/projects/web/app であることを確認します
  • ls で config.txt があることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • パスは projects/web/app のように / でつなぎます
  • 一つずつ cd projects → cd web → cd app でも到達できます
  • Tab キーでディレクトリ名を補完できます
