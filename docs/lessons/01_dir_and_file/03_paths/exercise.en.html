📋 Task

Move to the given paths with cd and confirm your location with pwd.

✏️ Instructions

  • Run cd documents to move using a relative path
  • Run pwd and confirm you are at /home/user/documents
  • Run cd .. to go up one level to /home/user
  • Run cd ~/documents to move using a ~ path
  • Run cd /home/user to return home using an absolute path
  • When ready, click the Test button

▶ 💡 Hints

  • Run pwd after each move to stay oriented
  • ~ is shorthand for /home/user
  • cd .. moves up one directory level
