Limits of a Plain Listing

In the previous lesson, you used ls to list the contents of a directory. With plain ls, you do not see every file.

What Are Hidden Files?

On Linux, files and directories whose names start with a dot (.) are called hidden files. For example, .config is a hidden file that often stores settings.

Hidden files are kept out of normal ls output so you do not accidentally delete or edit them. Many configuration files and auto-generated files use this naming style.

On Mac Finder, you can toggle hidden files with ⌘+Shift+. (period). In the terminal, ls -a serves the same purpose.

Example in the home directory (/home/user):

$ ls
documents
notes.txt

From this output alone, .config is invisible. But the actual structure may look like this:

/home/user/
├── .config          ← hidden file (not shown by plain ls)
├── documents/
└── notes.txt

ls -a — Show Hidden Files Too

-a stands for all. When you run ls -a, you see every entry, including hidden ones.

$ ls -a
.
..
.config
documents
notes.txt
  • . — a special name for the current directory itself
  • .. — a special name for the parent directory (one level up)
  • .config — a hidden file starting with a dot

. and .. are also used when specifying paths (you will learn more in the lesson "Understanding Paths"). For now, remember that they appear when you use ls -a.

ls -l — Detailed Listing

ls -l shows extra details such as file size, modification time, and the owner.

$ ls -l
-rw-r--r-- 1 user user  12 Jan  1 10:00 notes.txt
drwxr-xr-x 2 user user  64 Jan  1 10:00 documents

You will study what each column means (permissions, owner, and more) in Chapter 4. For now, it is enough to know that this richer view exists.

Tab Completion and Command History

The exercise terminal includes helpers for typing:

  • Tab key — after partial input, Tab completes command or file names (e.g. ls doc + Tab → ls documents)
  • ↑↓ keys — recall previously run commands so you do not retype them

Chapter 9 covers these in depth, but feel free to try them during exercises.
