この演習のゴール

散らばったファイルを目的別のフォルダに整理し、不要なファイルを片付けます。第2章で学んだ mkdir、mv、cp、rm を組み合わせて、実際の整理作業を体験します。

課題のイメージ

ホームディレクトリに散らばったファイルを、次のように整理します。

  • mkdir -p で docs と images フォルダを用意
  • mv でテキスト類を docs/、画像を images/ へ
  • cp で重要ファイルのコピーを残す
  • rm で不要ファイルを削除

作業の流れ

  1. ls で現状把握
  2. フォルダ作成 → 移動・コピー → 削除
  3. 再度 ls で結果確認

rm を使うときの注意

rm で削除したファイルは、ゴミ箱に入らず元に戻せません。削除する前に、次を確認してください。

  • ls で、本当に削除してよいファイルか
  • ファイル名を正確に入力しているか（打ち間違いに注意）

道場では リセット ボタンで最初からやり直せますが、本物の Linux では取り消せないので、削除は慎重に行いましょう。
