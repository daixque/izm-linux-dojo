総合演習：ファイル整理

第12章では、第1–9章で学んだコマンドを組み合わせて実践課題に取り組みます。まずは第2章のファイル操作の総仕上げです。

課題のイメージ

ホームディレクトリに散らばったファイルを、目的別のフォルダに整理します。

  • mkdir -p で docs と images フォルダを用意
  • mv でテキスト類を docs/、画像を images/ へ
  • cp で重要ファイルのコピーを残す
  • rm で不要ファイルを削除

作業の流れ

  1. ls で現状把握
  2. フォルダ作成 → 移動・コピー → 削除
  3. 再度 ls で結果確認

次のステップ

演習ページで、散らかったファイルを指定どおりに整理してください。
