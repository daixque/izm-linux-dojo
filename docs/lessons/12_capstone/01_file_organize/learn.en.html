Goal of This Exercise

Organize scattered files into purpose-specific folders and clean up unneeded files. You will combine mkdir, mv, cp, and rm from Chapter 2 in a real organizing task.

Task Overview

Organize files scattered in the home directory like this:

  • mkdir -p to create docs and images folders
  • mv text files into docs/, images into images/
  • cp to keep a copy of an important file
  • rm to remove unneeded files

Workflow

  1. ls to see the starting state
  2. Create folders → move/copy → delete
  3. ls again to verify

Caution When Using rm

Files deleted with rm do not go to a trash bin and cannot be easily restored. Before deleting, confirm:

  • With ls, that it is really the file you intend to delete
  • That the filename is typed correctly (watch for typos)

In the dojo you can use the Reset button to start over, but on real Linux deletion is permanent — proceed carefully.
