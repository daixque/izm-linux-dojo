Capstone: Organize Files

Chapter 12 combines commands from Chapters 1–9. First, a file-management capstone from Chapter 2.

Task Overview

Organize scattered files in the home directory into purpose-specific folders.

  • mkdir -p to create docs and images
  • mv text files into docs/, images into images/
  • cp to keep a copy of an important file
  • rm to remove unneeded files

Workflow

  1. ls to see the starting state
  2. Create folders → move/copy → delete
  3. ls again to verify
