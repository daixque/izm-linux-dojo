📋 課題内容

散らかったホームディレクトリを整理しましょう。

  1. docs と images フォルダを mkdir -p で作成
  2. report.txt と memo.txt を docs/ に移動
  3. inbox/letter.txt を docs/letter.txt にコピー
  4. photo.jpg を images/ に移動
  5. old_backup.txt を削除

✏️ やること

  • ls で最初の状態を確認します
  • 上記の手順を順番に実行します
  • ls docs と ls images で結果を確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • mv は元の場所からファイルがなくなります
  • cp 後も inbox/letter.txt は残ります
  • うまくいかないときは リセット ボタンで最初からやり直せます
