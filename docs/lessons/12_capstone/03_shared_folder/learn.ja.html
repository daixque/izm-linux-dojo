総合演習：共有フォルダ

第4–6章で学んだユーザー・権限・sudo を使い、複数人で使う共有ディレクトリを整備します。

シナリオ

/shared ディレクトリを、チームメンバー user が読み書きできる状態にします。初期状態では root オーナーで、一般ユーザーは書き込めません。

使うコマンド

  • sudo chown user:user /shared — オーナーとグループを user に変更
  • chmod u+rwx,go+rx /shared — user は読み書き実行、他は読み取りと実行
  • ls -l /shared — 変更結果を確認

なぜ sudo が必要？

他人（ここでは root）のファイルのオーナーを変える操作は、管理者権限が必要です。第6章で学んだ sudo を使います。
