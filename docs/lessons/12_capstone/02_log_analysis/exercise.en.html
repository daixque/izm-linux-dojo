📋 Task

Extract ERROR lines from app.log and count them.

  1. grep ERROR app.log to display ERROR lines
  2. grep ERROR app.log | wc -l to count (should be 3)
  3. grep ERROR app.log > errors.txt to save ERROR lines

✏️ Instructions

  • Run cat app.log to review the log
  • Run grep ERROR app.log
  • Run grep ERROR app.log | wc -l to count
  • Run grep ERROR app.log > errors.txt to save results
  • Click Test when ready

▶ 💡 Hints

  • Pipe | sends left output to the right command
  • wc -l counts lines
  • Use Reset to start over if needed
