Capstone: Log Analysis

Use grep, wc, and pipes (|) from Chapters 7–8 to extract information from log files.

What is a Log File?

Programs and servers write activity records to log files. INFO usually means normal operation, ERROR means failure, and WARN means warning.

Common Combinations

$ grep ERROR app.log
$ grep ERROR app.log | wc -l

The first shows only ERROR lines; the second counts how many ERROR lines exist.

Workflow

  1. Use cat or head to understand the log
  2. grep to extract ERROR lines
  3. Pipe to wc -l to count them
