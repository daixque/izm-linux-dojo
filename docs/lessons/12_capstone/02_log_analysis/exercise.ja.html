📋 課題内容

app.log から ERROR 行を抽出し、件数を数えましょう。

  1. grep ERROR app.log で ERROR 行を表示
  2. grep ERROR app.log | wc -l で件数を確認（3 件のはず）
  3. grep ERROR app.log > errors.txt で ERROR 行だけをファイルに保存

✏️ やること

  • cat app.log でログ全体を確認します
  • grep ERROR app.log を実行します
  • grep ERROR app.log | wc -l で件数を数えます
  • grep ERROR app.log > errors.txt で結果を保存します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • パイプ | は左の出力を右のコマンドに渡します
  • wc -l は行数を数えます
  • うまくいかないときは リセット ボタンで最初からやり直せます
