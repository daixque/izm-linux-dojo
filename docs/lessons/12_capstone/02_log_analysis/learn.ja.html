総合演習：ログ分析

第7–8章で学んだ grep、wc、パイプ（|）を使い、ログファイルから必要な情報を取り出します。

ログファイルとは

プログラムやサーバーは、動作の記録をログ（log）ファイルに書き込みます。INFO は通常動作、ERROR はエラー、WARN は警告を表すことが多いです。

よく使う組み合わせ

$ grep ERROR app.log
$ grep ERROR app.log | wc -l

1 行目は ERROR 行だけを表示、2 行目は ERROR の件数を数えます。

作業の流れ

  1. cat や head でログの全体像を把握
  2. grep で ERROR 行を抽出
  3. パイプで wc -l に渡して件数を数える

次のステップ

演習では app.log から ERROR の件数を集計します。
