📋 課題内容

通常はアクセスできない /root を sudo ls /root で一覧表示しましょう。

✏️ やること

  • まず ls /root を試すと Permission denied になることを確認します（任意）
  • sudo ls /root を実行します
  • secret.txt が表示されることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • sudo の後にスペースを入れてからコマンドを書きます
  • sudo 中だけ root 権限で実行されます
  • うまくいかないときは リセット ボタンで最初からやり直せます
