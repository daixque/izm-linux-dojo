📚 説明
English

root の権限を一時的に借りる

普通のユーザーは root の領域（/root など）にアクセスできません。でも、管理者の作業が必要なときがあります。

sudo（"superuser do"＝スーパーユーザーとして実行）は、一時的に root としてコマンドを実行する仕組みです。本物の Linux ではパスワードを求められますが、演習用シミュレータではパスワードなしで動作します。

使い方

$ ls /root
ls: cannot open directory '/root': Permission denied

$ sudo ls /root
secret.txt
backup/

sudo の後に、普通のコマンドをそのまま書きます。1 回のコマンドだけ root 権限で実行され、終わるとまた普通のユーザーに戻ります。

注意点

  • 必要なときだけ使う — 日常のファイル操作に sudo は不要
  • 間違えてシステムファイルを壊すと、PC 全体に影響する
  • macOS でも「管理者パスワード」を求める場面が sudo に近い
