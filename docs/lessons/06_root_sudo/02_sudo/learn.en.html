Borrowing root Power Temporarily

Normal users cannot access root-only areas like /root. Sometimes admin-level work is still needed.

sudo ("superuser do") runs a command as root for one shot. Real Linux asks for a password; this dojo simulates it without a password.

Usage

$ ls /root
ls: cannot open directory '/root': Permission denied

$ sudo ls /root
secret.txt
backup/

Put a normal command after sudo. Only that command runs as root; then you return to your normal user.

Cautions

  • Use it only when needed — daily file work does not need sudo
  • Mistakes on system files can affect the whole machine
  • On macOS, the admin password prompt is similar to sudo
