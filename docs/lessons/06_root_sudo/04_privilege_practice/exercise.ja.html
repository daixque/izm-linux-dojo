📋 課題内容

shared/data.txt のオーナーを user に変更し、権限を 664 に設定しましょう。

✏️ やること

  • cd shared に移動します
  • ls -l data.txt で現在の状態（root オーナー）を確認します
  • sudo chown user data.txt でオーナーを変更します
  • sudo chmod 664 data.txt で権限を設定します
  • ls -l data.txt で -rw-rw-r-- と user オーナーを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • 664 = オーナー rw、グループ rw、その他 r
  • sudo chown のあと sudo chmod の順が分かりやすいです
  • うまくいかないときは リセット ボタンで最初からやり直せます
