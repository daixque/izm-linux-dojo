📋 Task

Change shared/data.txt owner to user and set permissions to 664.

✏️ Instructions

  • Run cd shared
  • Run ls -l data.txt — currently owned by root
  • Run sudo chown user data.txt
  • Run sudo chmod 664 data.txt
  • Run ls -l data.txt — expect -rw-rw-r-- and owner user
  • When ready, click the Test button

▶ 💡 Hints

  • 664 = owner rw, group rw, others read
  • sudo chown then sudo chmod is a clear order
  • Use the Reset button to start over if needed
