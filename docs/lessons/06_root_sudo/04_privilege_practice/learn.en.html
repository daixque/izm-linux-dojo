Chapters 4–6 Summary

You have learned users, ownership, permissions, root, and sudo. In practice, you combine them.

Shared Folder Scenario

Imagine an admin setting up a /shared folder:

  1. Transfer ownership to user user (sudo chown)
  2. Set permissions so the team can read (sudo chmod)
  3. Verify with ls -l
$ sudo chown user:staff /shared/data.txt
$ sudo chmod 664 /shared/data.txt
$ ls -l /shared/data.txt
-rw-rw-r-- 1 user staff  ... data.txt

Mindset

  • sudo is a last resort — try normal methods first
  • Check ls -l before and after changes
  • Grant the minimum permissions needed
