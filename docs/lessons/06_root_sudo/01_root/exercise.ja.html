📋 課題内容

system ディレクトリで ls -l を実行し、root がオーナーの hosts ファイルを確認しましょう。

✏️ やること

  • cd system に移動します
  • ls -l でファイル一覧を表示します
  • hosts のオーナーが root であることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • root は管理者ユーザーです
  • 3列目（オーナー）が root のファイルはシステムファイルです
  • 普通のユーザーは root のファイルを変更できません
