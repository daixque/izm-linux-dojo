root（ルート）とは

Linux には root という特別なユーザーがいます。root はスーパーユーザー（管理者）と呼ばれ、システム上のほぼすべての操作ができます。

  • root のホームディレクトリは /root
  • システム設定は /etc など root が管理する領域にある
  • 普通のユーザー（user）は、root の領域には入れないことが多い

root オーナーのファイル

ls -l でオーナーが root のファイルをよく見かけます。

$ ls -l /etc
-rw-r--r-- 1 root root  123 Jan  1 10:00 hosts
drwxr-xr-x 2 root root 4096 Jan  1 10:00 config.d

普通のユーザーは、root のファイルを勝手に変更できません。システムを壊さないための大切な仕組みです。
