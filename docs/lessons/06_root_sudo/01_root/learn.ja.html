特別なユーザー root

Linux には user のような普通のユーザー以外に、root（ルート）という特別なユーザーがいます。root はスーパーユーザー（管理者）と呼ばれ、システム上のほぼすべての操作ができます。

  • root のホームディレクトリは /root
  • システム設定は /etc など root が管理する領域にある
  • 普通のユーザー（user）は、root の領域には入れないことが多い

システムを壊さないため、日常の作業は普通のユーザーで行い、必要なときだけ root の権限を使う、という使い分けが基本です。

root オーナーのファイル

ls -l でオーナーが root のファイルをよく見かけます。

$ ls -l /etc
-rw-r--r-- 1 root root  123 Jan  1 10:00 hosts
drwxr-xr-x 2 root root 4096 Jan  1 10:00 config.d

普通のユーザーは、root のファイルを勝手に変更できません。OS の設定を守るための大切な仕組みです。
