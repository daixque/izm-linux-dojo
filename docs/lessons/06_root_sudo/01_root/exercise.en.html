📋 Task

Run ls -l in the system directory and find the hosts file owned by root.

✏️ Instructions

  • Run cd system
  • Run ls -l to list files with details
  • Confirm hosts is owned by root
  • When ready, click the Test button

▶ 💡 Hints

  • root is the administrator user
  • Column 3 (owner) shows root for system files
  • Normal users cannot change root's files
