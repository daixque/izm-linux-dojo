What Is root?

Linux has a special user called root — the superuser (administrator) who can do almost anything on the system.

  • root's home directory is /root
  • System settings live in areas like /etc, managed by root
  • Normal users like user often cannot enter root-only areas

Files Owned by root

ls -l often shows files whose owner is root:

$ ls -l /etc
-rw-r--r-- 1 root root  123 Jan  1 10:00 hosts
drwxr-xr-x 2 root root 4096 Jan  1 10:00 config.d

Normal users cannot change root's files freely — that protects the system.
