オーナー変更

📚 説明
English

オーナーを変える chown

ファイルのオーナーを変更するコマンドが chown（change owner）です。グループだけ変える chgrp（change group）もあります。

$ chown user myfile.txt          # オーナーを user に
$ chown user:staff myfile.txt    # オーナーとグループを同時に
$ chgrp staff myfile.txt         # グループだけ staff に

なぜ sudo が必要か

他人がオーナーのファイルの chown / chgrp は、普通のユーザーにはできません。root だけが任意のオーナーに変更できます。

$ chown user admin.log
chown: changing ownership of 'admin.log': Operation not permitted

$ sudo chown user admin.log
# 成功

典型例

管理者が作ったファイルを、利用者に渡すとき — 「このファイルのオーナーを user に変更してね」と sudo chown で対応します。
