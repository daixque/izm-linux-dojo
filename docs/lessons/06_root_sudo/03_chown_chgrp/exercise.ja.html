📋 課題内容

admin.log のオーナーを sudo chown user admin.log で変更し、ls -l で確認しましょう。

✏️ やること

  • ls -l admin.log で現在のオーナー（root）を確認します
  • sudo chown user admin.log を実行します
  • ls -l admin.log でオーナーが user になったことを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • chown だけでは root のファイルは変更できません
  • sudo を付けると root として chown が実行されます
  • 変更後は cat もできるようになります
