Chapter 6 Lesson 3: Changing File Ownership

日本語

chown — Change Owner

chown changes a file's owner. chgrp changes only the group.

$ chown user myfile.txt          # owner → user
$ chown user:staff myfile.txt    # owner and group together
$ chgrp staff myfile.txt         # group → staff only

Why sudo Is Needed

Normal users cannot chown or chgrp files they do not own. Only root can set any owner.

$ chown user admin.log
chown: changing ownership of 'admin.log': Operation not permitted

$ sudo chown user admin.log
# succeeds

Typical Use

When an admin creates a file for a user — hand it over with sudo chown user ....
