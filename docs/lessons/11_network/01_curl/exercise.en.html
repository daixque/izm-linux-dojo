📋 Task

Fetch data from the mock API. First display the response with curl, then save to a file with -o.

✏️ Instructions

  • Run curl https://dojo.example/api/hello to display JSON
  • Run curl -o info.txt https://dojo.example/data/info.txt to save a file
  • Run cat info.txt to verify the saved content
  • Click Test when ready

▶ 💡 Hints

  • Put the filename right after -o
  • URLs start with https://dojo.example/...
  • Use Reset to start over if needed
