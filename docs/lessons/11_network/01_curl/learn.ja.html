curl で Web 取得

📚 説明
English

ネットワークと HTTP

インターネット上の Web サイトや API からデータを取得するとき、多くの場合 HTTP という決まりで通信します。ブラウザがページを表示するときも、裏では HTTP でデータを受け取っています。

ターミナルから Web のデータを取得する代表的なコマンドが curl です。道場では実際のインターネットには接続せず、固定の模擬レスポンスを返します。

curl URL — 内容を表示

$ curl https://dojo.example/api/hello
{"message":"Hello from Dojo API"}

URL を指定すると、サーバーから返ってきた内容（レスポンス）がターミナルに表示されます。

curl -o ファイル URL — ファイルに保存

-o オプションで、レスポンスをファイルに保存できます。第2章のファイル操作と組み合わせると便利です。

$ curl -o info.txt https://dojo.example/data/info.txt
$ cat info.txt
IZM Linux Dojo
Version 1.0

道場で使える URL

  • https://dojo.example/api/hello — JSON 形式の挨拶
  • https://dojo.example/data/info.txt — テキスト情報
  • https://dojo.example/api/status — ステータス JSON
