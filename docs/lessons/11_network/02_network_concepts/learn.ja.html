ネットワークコマンドの世界

Linux には、ネットワーク関連の便利なコマンドがたくさんあります。ここでは概念を押さえ、道場で実行できないものは「名前と役割」だけ覚えましょう。

wget — ファイルをダウンロード

curl と似ていますが、ファイルのダウンロードに特化したイメージです。URL を指定してファイルを保存します。道場では curl -o で同様の操作を学びました。

ping — 接続確認

指定したサーバーに「届きますか？」と問い合わせ、応答時間を表示します。ネットワークの疎通確認に使います。道場では実際のネットワーク通信は行いません。

ssh — リモートログイン

別のコンピューターに安全にログインして、ターミナル操作を行うコマンドです。サーバー管理でよく使われますが、設定や鍵が必要なため、道場の対象外です。

curl の復習

第11章で学んだ curl は、HTTP でデータを取得する基本ツールです。

  • curl URL — レスポンスを表示
  • curl -o ファイル URL — ファイルに保存

道場の模擬 URL：https://dojo.example/api/hello、https://dojo.example/data/info.txt
