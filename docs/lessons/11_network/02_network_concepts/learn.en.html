The World of Network Commands

Linux has many useful network commands. Here we focus on concepts. For commands the dojo cannot run, learn their names and roles.

wget — Download Files

Similar to curl, but often associated with downloading files. In the dojo you practiced the same idea with curl -o.

ping — Check Connectivity

Sends a "are you there?" message to a server and shows response time. Used to verify network connectivity. The dojo does not perform real network traffic.

ssh — Remote Login

Securely log in to another computer and use its terminal. Common in server administration, but requires setup and keys — outside the dojo scope.

Review: curl
