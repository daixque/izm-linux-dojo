📋 課題内容

前のレッスンの curl を復習しましょう。模擬 API からステータス JSON を取得し、status.json として保存します。

✏️ やること

  • curl https://dojo.example/api/status でステータスを表示します
  • curl -o status.json https://dojo.example/api/status でファイルに保存します
  • cat status.json で内容を確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • 前のレッスンと同じ -o の使い方です
  • JSON は { で始まるテキストデータです
  • うまくいかないときは リセット ボタンで最初からやり直せます
