chmod（記号形式）

📚 説明
English

権限を変更する chmod

オーナー（または後の章で学ぶ root）は、chmod コマンドで権限を変更できます。記号形式は、誰に・何を・どうするかを文字で指定する方法です。

記号の意味

  • 誰に（who） — u＝オーナー、g＝グループ、o＝その他、a＝すべて
  • 操作（operator） — +＝追加、-＝削除、=＝その値に設定
  • 権限（what） — r、w、x

よく使う例

$ chmod u+x script.sh     # オーナーに実行権限を追加
$ chmod go-r secret.txt   # グループとその他から読み取りを削除
$ chmod a+r public.txt    # すべてに読み取りを追加
$ chmod u=rw,go=r file    # オーナー: rw、グループとその他: r のみ

変更後は ls -l で確認する習慣をつけましょう。

注意

自分がオーナーでないファイルの chmod はできません（Operation not permitted）。他人のファイルの権限は、オーナーか管理者だけが変えられます。
