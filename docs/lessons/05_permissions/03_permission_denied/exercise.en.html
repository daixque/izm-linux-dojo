📋 Task

As in the lesson, try operations you are not allowed to do and confirm the errors. alice_secret.txt and alice_private belong to alice.

  • cat — no read permission → Permission denied
  • rm — no write permission → Permission denied
  • cd — no execute permission → Permission denied
  • chmod — not the owner → Operation not permitted

✏️ Instructions

  • Run ls -l to inspect alice_secret.txt and alice_private
  • Try cat alice_secret.txt and confirm Permission denied
  • Try rm alice_secret.txt and confirm Permission denied
  • Try cd alice_private and confirm Permission denied
  • Try chmod o+r alice_secret.txt and confirm Operation not permitted
  • Confirm your own my_notes.txt can be read with cat
  • When ready, click the Test button

▶ 💡 Hints

  • You are user, so you cannot read alice's files (cat)
  • Removing someone else's file with rm requires write permission on that file
  • Entering a directory with cd requires execute (x) permission
  • Only the owner (or root) can run chmod
  • These errors are expected — security is working
