What Is Permission Denied?

When you try an operation without permission, Linux returns Permission denied. It looks like an error, but it also means security is working.

Common Situations

  • cat on someone else's file (no read permission)
  • rm on someone else's file (no write permission)
  • cd into a directory without execute permission
  • chmod on a file you do not own
$ cat alice_secret.txt
cat: alice_secret.txt: Permission denied

What to Do

  1. Run ls -l to check owner and permissions
  2. On your own files, adjust with chmod (this chapter)
  3. For others' or system files, ask the owner or use sudo (Chapter 6) when appropriate

Do not loosen permissions or use sudo casually. First understand why you were blocked.
