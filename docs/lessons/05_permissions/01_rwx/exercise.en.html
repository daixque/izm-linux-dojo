📋 Task

Run ls -l and compare the permissions on public.txt and private.txt.

✏️ Instructions

  • Run ls -l for a detailed listing
  • public.txt has -rw-r--r-- (everyone can read)
  • private.txt has -rw------- (owner only)
  • When ready, click the Test button

▶ 💡 Hints

  • The first 10 characters are permissions
  • Characters 2–4 are the owner's (user's) rwx
  • Characters 8–10 are others' rwx
