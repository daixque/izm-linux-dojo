chmod（8進）【発展】

📚 説明
English

8進数（octal）による chmod

記号形式のほか、chmod では8進数（0〜7 の 3 桁）でも権限を指定できます。上級者向けですが、644 や 755 という数字をよく見かけるので、意味を知っておくと便利です。

3 桁の意味

3 桁それぞれが、オーナー・グループ・その他の rwx を数字 1 つで表します。

  • 4＝r、2＝w、1＝x（足し算で組み合わせ）
  • 7＝4+2+1＝rwx、6＝rw、5＝rx、4＝r
644  →  rw- r-- r--   （ファイルによく使う: オーナー読書、その他読のみ）
755  →  rwx r-x r-x   （ディレクトリや実行ファイルによく使う）
600  →  rw- --- ---   （オーナーだけ読書）

使い方

$ chmod 644 report.txt
$ chmod 755 myproject/

記号形式と 8 進は、同じ権限を違う書き方で指定しているだけです。どちらか慣れた方を使えば OK です。
