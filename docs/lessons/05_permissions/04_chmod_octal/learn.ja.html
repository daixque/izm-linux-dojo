chmod（8進）

📚 説明
English

数字で権限を指定する方法

記号形式のほか、chmod では8進数（0〜7 の 3 桁）でも権限を指定できます。644 や 755 という数字をよく見かけるので、意味を知っておくと便利です。

3 桁の意味

3 桁それぞれが、オーナー・グループ・その他の rwx を数字 1 つで表します。

  • 4＝r（読む）、2＝w（書く）、1＝x（実行）
  • 足し算で組み合わせます — 7＝4+2+1＝rwx、6＝rw、5＝rx、4＝r

よく使う数字

8進数権限文字列用途の例
644rw- r-- r--普通のテキストファイル（オーナーは読書、その他は読みのみ）
755rwx r-x r-xディレクトリや実行ファイル
600rw- --- ---オーナーだけが読書できるファイル

使い方

$ chmod 644 report.txt
$ chmod 755 myproject/

記号形式と 8 進数は、同じ権限を違う書き方で指定しているだけです。どちらか慣れた方を使えば OK です。
