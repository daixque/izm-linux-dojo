📋 Task

Set report.txt to 644 and the tools directory to 755.

✏️ Instructions

  • Run chmod 644 report.txt
  • Run chmod 755 tools
  • Run ls -l to verify the permissions
  • When ready, click the Test button

▶ 💡 Hints

  • 644 is rw-r--r--, 755 is rwxr-xr-x
  • chmod works on directories too
  • Use the Reset button to start over if needed
