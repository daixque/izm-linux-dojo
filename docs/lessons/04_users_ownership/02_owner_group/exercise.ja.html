📋 課題内容

ls -l を実行し、memo.txt のオーナーが user であることを確認しましょう。

✏️ やること

  • ls -l で詳細な一覧を表示します
  • memo.txt の行を探します
  • 3列目（オーナー）と 4列目（グループ）が user であることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • ls と -l の間にスペースを入れます
  • 3列目がオーナー、4列目がグループです
  • documents はディレクトリなので先頭が d になります
