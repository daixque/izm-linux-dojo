オーナーシップによる違いを確認する

📚 説明
English

自分のファイルと他人のファイル

ファイルを触る前に、次の 2 点を確認する習慣をつけましょう。

  1. whoami で自分のユーザー名を確認する
  2. ls -l で対象ファイルのオーナー（3 列目）を確認する

オーナーが whoami と同じ → だいたい自分のファイル（自由に触れることが多い）
オーナーが違う → 他人のファイル（制限があることが多い）

この章のポイント

コマンド・概念内容
whoami今のユーザー名を確認
ls -l の 3 列目オーナー（持ち主）
ls -l の 4 列目グループ
複数ユーザー同じフォルダに他人のファイルが混在することがある

次に見ること — 権限

オーナー以外にも、「グループ」や「その他のユーザー」に許可を与える仕組みがあります。それが権限（permissions）です。

他人のファイルで cat が失敗したり、touch が拒否されたりするのは、権限のせいです。次の章では rwx の読み方と chmod による変更方法を扱います。
