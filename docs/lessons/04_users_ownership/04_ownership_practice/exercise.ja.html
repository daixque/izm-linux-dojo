📋 課題内容

workspace フォルダで whoami と ls -l を使い、自分がオーナーの my_draft.txt の内容を cat で表示しましょう。

✏️ やること

  • cd workspace に移動します
  • whoami で自分のユーザー名を確認します
  • ls -l で各ファイルのオーナーを確認します
  • my_draft.txt は自分（user）のファイルです — cat my_draft.txt で内容を表示します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • other_person.txt のオーナーは alice — 他人のファイルです
  • オーナーが user のファイルだけ自由に cat できます（第5章で詳しく）
  • うまくいかないときは リセット ボタンで最初からやり直せます
