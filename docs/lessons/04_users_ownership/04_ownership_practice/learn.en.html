Chapter 4 Summary

In this chapter you learned:

  • whoami — check your current username
  • ls -l — read owner (column 3) and group (column 4)
  • Multiple users — others' files can appear in the same folder

Your Files vs Others' Files

Before operating on a file, make a habit of:

  1. Running whoami to see your username
  2. Running ls -l to see the file's owner

Owner matches whoami → usually your file (you can often change it freely)
Owner differs → someone else's file (restrictions often apply)

Preview of Chapter 5

Besides the owner, Linux grants access to the group and everyone else through permissions.

When cat fails on another user's file or touch is refused, permissions are why. Next chapter covers rwx and chmod.
