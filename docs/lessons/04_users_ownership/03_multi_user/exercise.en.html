📋 Task

Move into the shared directory and use ls -l to see who owns each file.

✏️ Instructions

  • Run cd shared to enter the shared directory
  • Run ls -l to list files with details
  • my_report.txt is owned by user (you)
  • alice_notes.txt and bob_data.txt belong to others
  • When ready, click the Test button

▶ 💡 Hints

  • Compare whoami with column 3 of ls -l
  • Same owner name means it is your file
  • Use the Reset button to start over if needed
