The Problem with Very Long Files

When you cat a long file, the text scrolls off the screen before you can read it all. On real Linux systems, less and more let you read one screen at a time.

$ less story.txt
(one screen is shown; press Space for the next page)

Press q to quit less. Arrow keys let you scroll line by line.

In This Dojo

The browser simulator does not fully reproduce less paging. Learn the concept here, and use head and tail for practice.

Practical Ways to Inspect Long Files

  • See the beginning — head -n 10 filename
  • See the end — tail -n 5 filename
  • Know the total length — wc -l filename

For a 100-line log, you might use head to learn the format and tail to check recent errors — a common real-world pattern.
