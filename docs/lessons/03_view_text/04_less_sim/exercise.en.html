📋 Task

story.txt is a very long file. Instead of cating the whole thing, inspect only what you need:

  • head -n 3 story.txt — the opening 3 lines
  • tail -n 3 story.txt — the final 3 lines

✏️ Instructions

  • Optionally run wc -l story.txt to see total line count
  • Run head -n 3 story.txt to show the beginning
  • Run tail -n 3 story.txt to show the ending
  • When ready, click the Test button

▶ 💡 Hints

  • Reading part of a long file with head/tail is a core skill
  • less pages through files on real Linux; here we use head/tail
  • Use the Reset button to start over if needed
