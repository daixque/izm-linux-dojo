📋 課題内容

story.txt はとても長いファイルです。cat ですべて表示するのではなく、次の操作で必要な部分だけ確認しましょう。

  • head -n 3 story.txt で冒頭 3 行（タイトル部分）を見る
  • tail -n 3 story.txt で末尾 3 行（結末部分）を見る

✏️ やること

  • wc -l story.txt で全体の行数を確認してもよいでしょう（任意）
  • head -n 3 story.txt で冒頭を表示します
  • tail -n 3 story.txt で結末を表示します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • 長いファイルは head/tail で部分的に読むのが基本テクニックです
  • less は本物の Linux ではページ送りできますが、この演習では head/tail を使います
  • うまくいかないときは リセット ボタンで最初からやり直せます
