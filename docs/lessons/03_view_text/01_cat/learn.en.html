Reading File Contents

In Chapter 2 you learned to create, copy, move, and delete files. Now you will learn to read text inside files from the terminal.

Many Linux config files and logs are plain text. Being able to read them helps with troubleshooting and understanding settings.

cat — Show the Whole File

cat (concatenate) prints a file's contents to the screen. Use it for short text files.

$ cat messages.txt
Hello!
Welcome to Linux Dojo.

Pass the filename as an argument:

cat filename

Multiple Files

You can list several files and cat will show them one after another:

$ cat part1.txt part2.txt

Not Ideal for Long Files

cat prints the entire file at once. For very long files (like logs), the output scrolls off the screen. Later lessons cover head, tail, and the idea of less for large files.

Combine with Chapter 1

You can cd into the directory first, or use a path: cat documents/readme.txt.
