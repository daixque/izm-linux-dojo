View Part of a Long File

cat from the previous lesson shows the entire file. For long files like logs, that overflows the screen.

Instead, use commands that show only the beginning or end of a file.

head — Show the Beginning

head prints the first lines of a file. The default is 10 lines.

$ head server.log
2024-01-01 INFO Server started
2024-01-01 INFO Listening on port 8080
...

Choose Line Count — head -n

The -n option sets how many lines to show:

$ head -n 5 server.log

tail — Show the End

tail prints the last lines. Useful for checking the newest log entries.

$ tail server.log
...
2024-01-04 INFO Goodbye

Choose Line Count — tail -n

$ tail -n 3 server.log

When to Use Which

  • Learn the file format → head
  • Check recent errors → tail
