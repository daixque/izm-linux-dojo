📋 Task

For server.log, run both of the following:

  • Show the first 5 lines with head -n 5
  • Show the last 3 lines with tail -n 3

✏️ Instructions

  • Run ls to confirm server.log exists
  • Run head -n 5 server.log to show the first 5 lines
  • Run tail -n 3 server.log to show the last 3 lines
  • When ready, click the Test button

▶ 💡 Hints

  • Order is: -n, then the number, then the filename
  • The newest log entries are usually at the end
  • Use the Reset button to start over if needed
