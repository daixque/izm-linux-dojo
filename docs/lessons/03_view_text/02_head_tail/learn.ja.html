先頭・末尾

📚 説明
English

長いファイルの一部だけ見る

前のレッスンで学んだ cat はファイル全体を表示します。ログファイルのように行数が多いファイルでは、全部を cat しても画面からあふれてしまいます。

そこで先頭や末尾の数行だけを取り出すコマンドを使います。

head — 先頭を表示

head はファイルの最初の方の行を表示します。デフォルトでは 10 行です。

$ head server.log
2024-01-01 INFO Server started
2024-01-01 INFO Listening on port 8080
...

行数を指定する — head -n

-n オプションで表示する行数を変えられます。

$ head -n 5 server.log

先頭 5 行だけが表示されます。

tail — 末尾を表示

tail はファイルの最後の方の行を表示します。ログの最新エントリを確認するときによく使います。

$ tail server.log
...
2024-01-04 INFO Goodbye

行数を指定する — tail -n

$ tail -n 3 server.log

末尾 3 行だけが表示されます。

使い分けの例

  • ファイルの形式を知りたい → head で先頭数行を見る
  • 最新のエラーを確認したい → tail で末尾数行を見る
