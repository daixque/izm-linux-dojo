先頭・末尾

📚 説明
English

長いファイルの一部だけ見る

cat はファイル全体を表示します。ログファイルのように行数が多いファイルでは、全部を表示しても画面からあふれてしまい、必要な部分を見つけにくくなります。

そんなときは、先頭や末尾の数行だけを取り出して確認する方法が便利です。

head — 先頭を表示する

head は、ファイルの最初の方の行を表示します。何も指定しなければ 10 行です。

$ head server.log
2024-01-01 INFO Server started
2024-01-01 INFO Listening on port 8080
...

ファイルの書式や項目名を確認したいときによく使います。

行数を指定する — head -n

-n オプションで、表示する行数を変えられます。

$ head -n 5 server.log

先頭 5 行だけが表示されます。

tail — 末尾を表示する

tail は、ファイルの最後の方の行を表示します。ログの最新の記録を確認するときによく使います。

$ tail server.log
...
2024-01-04 INFO Goodbye

何も指定しなければ、末尾 10 行が表示されます。

行数を指定する — tail -n

$ tail -n 3 server.log

末尾 3 行だけが表示されます。

使い分けの例

  • ファイルの形式を知りたい → head で先頭数行を見る
  • 最新の記録を確認したい → tail で末尾数行を見る

たとえばログファイルなら、head で項目の並びを確認し、tail で直近のエラーやメッセージを確認する、という使い方が現場でもよくあります。
