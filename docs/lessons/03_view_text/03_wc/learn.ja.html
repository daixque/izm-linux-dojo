第3章 レッスン3: ファイルの行数や語数を数える

📚 説明
English

ファイルの「量」を数える

テキストファイルは、中身を読むだけでなく「何行あるか」「どれくらいの長さか」も知りたい場面があります。レポートの行数を数えたり、ログがどれだけ増えたか調べたりするときに便利です。

Linux では wc（word count）というコマンドで、ファイルの量を数えられます。

wc -l — 行数を数える

-l オプション（line＝行）で、ファイルの行数を数えます。

$ wc -l essay.txt
3 essay.txt

左側の数字が行数です。この例では 3 行あります。改行で区切られた 1 かたまりが 1 行として数えられます。

wc -w — 語数を数える

-w オプション（word＝語）で、空白で区切られた単語の数を数えます。

$ wc -w essay.txt
8 essay.txt

英語の文章では単語数、日本語では空白の有無によって数え方が変わることがあります。

wc -c — バイト数を数える

-c オプション（character / byte＝文字・バイト）で、ファイルのサイズ（バイト数）を数えます。英字中心の短いテキストでは、おおよそ文字数に近い値になります。

$ wc -c essay.txt
52 essay.txt

よく使うオプションのまとめ

オプション数えるもの
-l行数
-w語数（空白区切りのかたまり）
-cバイト数（ファイルサイズ）

オプションを付けない wc は、行数・語数・バイト数をまとめて表示します。まずは -l から覚えておけば十分です。
