行数・語数

📚 説明
English

ファイルの量を数える

テキストファイルを読むだけでなく、「何行あるか」「どれくらいの長さか」を知りたい場面もあります。レポートの行数を数えたり、ログがどれだけ増えたか調べたりするときに便利です。

Linux では wc（word count）コマンドを使います。

wc -l — 行数を数える

-l オプション（lines）で、ファイルの行数を数えます。

$ wc -l essay.txt
3 essay.txt

「3 行ある」という意味です。改行で区切られた行が 1 行として数えられます。

wc -w — 語数を数える

-w オプション（words）で、空白で区切られた単語の数を数えます。

$ wc -w essay.txt
8 essay.txt

wc -c — バイト数を数える

-c オプション（bytes）で、ファイルの文字数（バイト数）を数えます。英字中心のテキストではおおよそ文字数と同じと考えてよいです。

$ wc -c essay.txt
52 essay.txt

オプションなしの wc

オプションを付けないと、行数・語数・バイト数をまとめて表示します（上級者向け）。この道場では -l・-w・-c を中心に学びます。
