📋 Task

For essay.txt, run the following:

  • wc -l essay.txt to count lines
  • wc -w essay.txt to count words

✏️ Instructions

  • Optionally run cat essay.txt to read the contents first
  • Run wc -l essay.txt
  • Run wc -w essay.txt
  • When ready, click the Test button

▶ 💡 Hints

  • Line count corresponds to newline characters
  • Word count splits on whitespace
  • Use the Reset button to start over if needed
