Counting File Contents

Sometimes you need to know how much text a file contains — how many lines, words, or bytes. This helps when checking report length or log growth.

On Linux you use wc (word count).

wc -l — Count Lines

The -l option (lines) counts how many lines the file has.

$ wc -l essay.txt
3 essay.txt

Each line separated by a newline counts as one line.

wc -w — Count Words

The -w option (words) counts words separated by whitespace.

$ wc -w essay.txt
8 essay.txt

wc -c — Count Bytes

The -c option (bytes) counts the file size in bytes. For mostly English text, this is roughly the character count.

$ wc -c essay.txt
52 essay.txt

wc Without Options

With no options, wc shows lines, words, and bytes together (advanced). This dojo focuses on -l, -w, and -c.
