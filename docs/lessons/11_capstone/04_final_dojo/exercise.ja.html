📋 課題内容

最終試練です。次の操作を順番に完了してください。

  1. mkdir -p project/logs でフォルダを作成
  2. cp server.log project/logs/ でログをコピー
  3. grep ERROR project/logs/server.log > project/errors.txt で ERROR を保存
  4. wc -l project/errors.txt で行数を確認（2 行）
  5. echo "Project complete" > project/README.txt で README を作成

✏️ やること

  • ls で server.log があることを確認します
  • 上記 5 ステップを順番に実行します
  • ls project で README.txt と errors.txt があることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • > は grep の結果をファイルに書き込みます（第8章）
  • 最後に cat project/README.txt で確認してもよいです
  • うまくいかないときは リセット ボタンで最初からやり直せます
