📋 Task

Final challenge — complete these steps in order:

  1. mkdir -p project/logs to create folders
  2. cp server.log project/logs/ to copy the log
  3. grep ERROR project/logs/server.log > project/errors.txt to save ERROR lines
  4. wc -l project/errors.txt to count lines (should be 2)
  5. echo "Project complete" > project/README.txt to create README

✏️ Instructions

  • Run ls and confirm server.log exists
  • Complete all 5 steps in order
  • Run ls project and confirm README.txt and errors.txt exist
  • Click Test when ready

▶ 💡 Hints

  • > writes grep output to a file (Chapter 8)
  • Optionally run cat project/README.txt at the end
  • Use Reset to start over if needed
