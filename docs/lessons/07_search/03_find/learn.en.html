Search by Filename

grep searches file contents. When you want to find files by name or location, use the find command.

For example, "list every .txt file somewhere under this folder" is a common use case. It is similar to searching inside a folder in Mac Finder.

Basic Form

find DIRECTORY -name PATTERN
  • DIRECTORY — where to start searching (. means the current directory)
  • -name — option to search by filename
  • PATTERN — the filename shape you want (wildcards like * work)

Example: Find All .txt Files

$ find . -name "*.txt"
./notes.txt
./documents/readme.txt

* means "any string." *.txt means "names ending in .txt."

Searching Subdirectories Too

find walks through folders inside the directory you specify and searches below them as well. For example, it can find ./documents/readme.txt even when it is deep in the tree.

ls shows only one level of the current directory. find goes down through the folder hierarchy to search.
