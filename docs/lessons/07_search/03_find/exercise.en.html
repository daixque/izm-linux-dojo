📋 Task

Find all files ending in .txt under the current directory.

✏️ Instructions

  • Run ls to see the home directory structure
  • Run find . -name "*.txt"
  • Confirm that three .txt files are found
  • Click the Test button when ready

▶ 💡 Hints

  • Quote the pattern: "*.txt"
  • . means the current directory
