Search Inside Text

In Chapter 3 you learned to read file contents with cat, head, and tail. Logs and config files can have many lines, and you often need to find specific lines quickly.

For example, in an error log you may want only the lines containing ERROR. Scanning line by line is tedious, so there is a way to automatically show only lines that contain a given string.

What Is grep?

grep is a command that extracts and displays lines containing a specified string from a file. It is used to search text files.

Basic Form

grep PATTERN FILENAME

Write the string you want to find as the pattern. Matching is case-sensitive (ERROR and error are treated as different).

$ grep ERROR app.log
2024-01-02 ERROR Connection failed
2024-01-03 ERROR Disk full

In the example above, each line of app.log is checked, and only lines containing ERROR are shown.

Search Multiple Files

You can list several filenames and grep will search each one in order.

$ grep ERROR log1.txt log2.txt

When Nothing Matches

If no line matches the pattern, grep prints nothing and exits. This is not an error — it means no matching lines were found.
