パターンでファイル名を指定する

📚 説明
English

ファイル名の一部をざっくり指定する

前のレッスンで find -name "*.txt" を使いました。この * のような記号をワイルドカードと呼びます。ファイル名の一部を「ここは何でもよい」とざっくり指定できます。

シェルはコマンドを実行する前に、ワイルドカードを実際のファイル名に置き換えます。この置き換えを「展開」と呼びます。

主なワイルドカード

記号意味例
*任意の文字列（0 文字以上）*.txt → notes.txt, readme.txt
?任意の 1 文字file?.txt → file1.txt, fileA.txt

ls *.txt の例

$ ls *.txt
notes.txt  readme.txt

シェルは *.txt を、今いるディレクトリで .txt で終わるファイル名に置き換えてから ls を実行します。実際には ls notes.txt readme.txt と同じ意味になります。

ワイルドカードと find の違い

方法動き
ワイルドカードシェルが展開する。主に今いるディレクトリの 1 階層で使う
find指定したフォルダの下の階層すべてをたどって探す

今いるフォルダだけなら ls *.txt で十分です。下のフォルダも含めて探すときは find を使います。
