📋 Task

Stop the sleep 100 process (PID 3099) with kill, then confirm it no longer appears in ps.

✏️ Instructions

  • Run ps and confirm PID 3099 exists
  • Run kill 3099
  • Run ps again and confirm 3099 is gone
  • Click Test when ready

▶ 💡 Hints

  • Type a space after kill before the PID
  • A successful kill produces no output
  • Use Reset to start over if needed
