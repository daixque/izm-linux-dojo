プロセス一覧

English

プロセスとは？

コンピューター上で動いているプログラムのことをプロセス（process）と呼びます。たとえば、ターミナルで bash を起動しているとき、Web ブラウザを開いているとき、裏で更新プログラムが動いているとき — それぞれがプロセスです。

第1章で学んだファイルは「保存されているデータ」、プロセスは「今まさに実行されている仕事」と考えると区別しやすいです。

プロセス ID（PID）

OS は動いているプロセスを区別するために、それぞれにプロセス ID（Process ID、略して PID）という番号を付けます。PID は数字だけで、たとえば 1042 や 3099 のようになります。

  • 同じプログラムを 2 回起動しても、PID は別々
  • プロセスが終了すると、その PID は別のプロセスに再利用されることがある

ps — プロセス一覧を表示

ps コマンドで、現在動いているプロセスの一覧を確認できます。道場のシミュレータでは、学習用の固定データが表示されます。

$ ps
PID TTY          TIME CMD
  1 ?           00:00:01 systemd
1042 pts/0       00:00:00 bash
2048 pts/0       00:00:02 python simulator.py
3099 pts/0       00:00:00 sleep 100

主な列の意味：

  • PID — プロセス ID
  • TTY — どの端末に接続しているか（? は端末なし）
  • TIME — CPU を使った時間の目安
  • CMD — 実行中のコマンド名

top について（参考）

本物の Linux では top コマンドで、プロセス一覧をリアルタイムに更新しながら表示できます。道場では再現が難しいため、ps で一覧を確認する方法を中心に学びます。
