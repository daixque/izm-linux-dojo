What is a Process?

A process is a program that is currently running on the computer. When you use a terminal with bash, open a web browser, or run a background update — each is a process.

Think of files (from Chapter 1) as stored data, and processes as work happening right now.

Process ID (PID)

The OS assigns each running process a number called a Process ID (PID), such as 1042 or 3099.

  • Starting the same program twice gives two different PIDs
  • When a process exits, its PID may later be reused

ps — List Processes

The ps command shows running processes. In the dojo simulator, fixed sample data is displayed.

$ ps
PID TTY          TIME CMD
  1 ?           00:00:01 systemd
1042 pts/0       00:00:00 bash
2048 pts/0       00:00:02 python simulator.py
3099 pts/0       00:00:00 sleep 100

Column meanings:

  • PID — process ID
  • TTY — terminal attached (? means none)
  • TIME — approximate CPU time used
  • CMD — command name

About top (Reference)

On real Linux, top shows a live-updating process list. The dojo focuses on ps instead.
