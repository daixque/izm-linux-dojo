📋 課題内容

ターミナルで次の操作を行い、ファイルとフォルダを新しく作ってみましょう。

  • workspace ディレクトリを作成する
  • project/src を mkdir -p で一度に作成する
  • memo.txt という空ファイルを作成する

✏️ やること

  • pwd で /home/user にいることを確認します
  • mkdir workspace で workspace フォルダを作ります
  • mkdir -p project/src で project と src を一度に作ります
  • touch memo.txt で空ファイルを作ります
  • ls で memo.txt、project、workspace が表示されることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • コマンドのあとに Enter キーを押します
  • Tab キーでコマンド名やパスを補完できます
  • うまくいかないときは リセット ボタンで最初からやり直せます
