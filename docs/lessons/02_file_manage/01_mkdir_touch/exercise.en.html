📋 Task

Create the following items in the terminal:

  • A directory named project
  • A nested directory project/src using mkdir -p
  • An empty file named memo.txt

✏️ Instructions

  • Run pwd to confirm you are in /home/user
  • Run mkdir project to create the project folder
  • Run mkdir -p project/src to add the src folder inside project
  • Run touch memo.txt to create an empty file
  • Run ls and confirm memo.txt and project appear
  • When ready, click the Test button

▶ 💡 Hints

  • Press Enter after each command
  • Use Tab to autocomplete command names and paths
  • Use the Reset button to start over if needed
