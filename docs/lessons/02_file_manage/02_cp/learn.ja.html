English

コピーとは

ファイルやフォルダのコピーは、元のデータを残したまま、同じ内容の別コピーを作る操作です。1 冊の本をコピー機で複写するイメージで、元と写しの 2 つが存在します。

Mac の Finder では、ファイルを選んで ⌘+C（コピー）→ 移動先で ⌘+V（貼り付け）が同じ操作です。Linux では cp（copy）コマンドを使います。

ファイルをコピーする — cp

基本形は次のとおりです。

cp コピー元 コピー先

たとえば report.txt を backup フォルダの中にコピーするには:

$ cp report.txt backup/
$ ls backup
report.txt

元の report.txt はそのまま残り、backup/report.txt という新しいファイルができます。

別名でコピーする

コピー先にファイル名を指定すると、別名のコピーも作れます。

$ cp report.txt report_backup.txt

ディレクトリをコピーする — cp -r

フォルダをコピーするときは -r オプションが必要です。フォルダの中身（子フォルダも含めて）ごとまとめてコピーします。

$ cp -r photos photos_copy
$ ls
photos
photos_copy

-r を付け忘れると「ディレクトリはコピーできない」というエラーになることが多いです。

コピー後の確認

第1章の ls や第3章で学ぶ cat を使って、コピー先にファイルがあるか確認しましょう。第1章の cd でコピー先フォルダに移動してから ls する方法もあります。
