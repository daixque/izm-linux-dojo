📋 課題内容

report.txt を backup フォルダの中にコピーしましょう。元のファイルはそのまま残ります。

✏️ やること

  • ls で report.txt と backup があることを確認します
  • cp report.txt backup/ でファイルをコピーします
  • ls backup で report.txt が表示されることを確認します
  • cat backup/report.txt で内容が同じか確認してもよいでしょう
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • コピー先のフォルダ名の末尾に / を付けると「フォルダの中」と分かりやすいです
  • 元の report.txt も ls に残っているはずです
  • うまくいかないときは リセット ボタンで最初からやり直せます
