📋 Task

Copy report.txt into the backup folder. The original file should remain in place.

✏️ Instructions

  • Run ls to confirm report.txt and backup exist
  • Run cp report.txt backup/ to copy the file
  • Run ls backup and confirm report.txt appears
  • Optionally run cat backup/report.txt to check the contents match
  • When ready, click the Test button

▶ 💡 Hints

  • Adding / after the folder name makes it clear you mean inside the folder
  • The original report.txt should still appear in ls
  • Use the Reset button to start over if needed
