📋 課題内容

散らかったホームディレクトリを整理しましょう。次の操作を順番に行います。

  1. work/archive フォルダを mkdir -p で作成
  2. notes.txt を work/ にコピー
  3. draft.txt を work/archive/ に移動
  4. trash/old.txt を削除し、空になった trash フォルダも削除

✏️ やること

  • ls で最初の状態（draft.txt, notes.txt, trash/, work/）を確認します
  • mkdir -p work/archive でアーカイブ用フォルダを作ります
  • cp notes.txt work/ で notes.txt を work にコピーします
  • mv draft.txt work/archive/ で draft.txt を移動します
  • rm trash/old.txt で不要ファイルを削除します
  • rmdir trash で空の trash フォルダを削除します
  • ls work と ls work/archive で結果を確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • 各ステップのあとに ls で確認すると安心です
  • cp 後も元の notes.txt はホームに残ります
  • うまくいかないときは リセット ボタンで最初からやり直せます
