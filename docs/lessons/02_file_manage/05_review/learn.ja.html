第1章 — 場所の確認と移動

コマンド意味
pwd今いるディレクトリのパスを表示
lsカレントディレクトリの中身を一覧
cd別のディレクトリへ移動

第2章 — ファイルの作成と管理

コマンド意味
mkdirディレクトリを作成
mkdir -p親ディレクトリごと階層を作成
touch空ファイルを作成
cpファイルをコピー（元は残る）
cp -rディレクトリをコピー
mv移動・リネーム（元はなくなる）
rmファイルを削除
rm -r / rmdirディレクトリを削除

操作の流れを意識する

実際の作業では、次のような流れでコマンドを組み合わせます。

  1. pwd と ls で現状を把握する
  2. mkdir -p で必要なフォルダを用意する
  3. cp や mv でファイルを整理する
  4. rm で不要なものを片付ける
  5. もう一度 ls で結果を確認する

よくある間違い

  • 場所を間違える — 作業前に pwd を確認
  • cp と mv を混同する — コピーは残る、移動は消える
  • 削除を急ぐ — ファイル名とパスを必ず読む
