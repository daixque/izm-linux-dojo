📋 Task

Organize the messy home directory by completing these steps in order:

  1. Create work/archive with mkdir -p
  2. Copy notes.txt into work/
  3. Move draft.txt into work/archive/
  4. Delete trash/old.txt and remove the empty trash folder

✏️ Instructions

  • Run ls to see the starting state (draft.txt, notes.txt, trash/, work/)
  • Run mkdir -p work/archive to create the archive folder
  • Run cp notes.txt work/ to copy notes.txt into work
  • Run mv draft.txt work/archive/ to move draft.txt
  • Run rm trash/old.txt to delete the unneeded file
  • Run rmdir trash to remove the empty trash folder
  • Run ls work and ls work/archive to verify
  • When ready, click the Test button

▶ 💡 Hints

  • Run ls after each step to stay on track
  • After cp, the original notes.txt still remains in home
  • Use the Reset button to start over if needed
