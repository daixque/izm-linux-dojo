Move and Rename

mv (move) moves a file or directory to a new location — like Cut and Paste in a GUI.

It also works for renaming when source and destination are in the same folder.

Rename a File

$ mv old_name.txt new_name.txt
$ ls
new_name.txt

The filename changes from old_name.txt to new_name.txt in the same directory.

Move to Another Folder

$ mv new_name.txt archive/
$ ls archive
new_name.txt

The file moves into archive and disappears from its original location.

Move and Rename at Once

$ mv draft.txt archive/final.txt

draft.txt becomes archive/final.txt in one step.

cp vs mv

  • cp — keeps the original and creates a copy
  • mv — removes the item from the old location
