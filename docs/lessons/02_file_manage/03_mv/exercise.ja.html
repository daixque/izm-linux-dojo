📋 課題内容

old_name.txt の名前を new_name.txt に変え、その後 archive フォルダへ移動しましょう。

✏️ やること

  • ls で old_name.txt と archive があることを確認します
  • mv old_name.txt new_name.txt で名前を変更します
  • ls で new_name.txt だけが表示されることを確認します
  • mv new_name.txt archive/ で archive フォルダへ移動します
  • ls archive で new_name.txt があることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • mv の第1引数が移動元、第2引数が移動先です
  • 移動後、元の場所からファイルが消えるのが正常です
  • うまくいかないときは リセット ボタンで最初からやり直せます
