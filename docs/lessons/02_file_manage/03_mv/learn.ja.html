移動と名前変更

mv（move）は、ファイルやディレクトリを別の場所へ移動するコマンドです。GUI の「切り取りして貼り付け」に近い操作です。

実は mv は名前の変更（リネーム）にも使えます。同じフォルダの中で名前だけ変えるときも mv です。

名前を変える — リネーム

$ mv old_name.txt new_name.txt
$ ls
new_name.txt

同じディレクトリ内で、ファイル名だけが old_name.txt から new_name.txt に変わります。

別のフォルダへ移動する

$ mv new_name.txt archive/
$ ls archive
new_name.txt

ファイルが archive フォルダの中に移動します。元の場所からは消えます。

移動と名前変更を同時に

移動先で別名を付けることもできます。

$ mv draft.txt archive/final.txt

これで draft.txt は archive/final.txt という名前で移動します。

cp との違い

  • cp — 元を残してコピーを作る
  • mv — 元の場所からなくなり、新しい場所（または新しい名前）に移る
