📋 Task

Rename old_name.txt to new_name.txt, then move it into the archive folder.

✏️ Instructions

  • Run ls to confirm old_name.txt and archive exist
  • Run mv old_name.txt new_name.txt to rename the file
  • Run ls and confirm only new_name.txt appears (not old_name.txt)
  • Run mv new_name.txt archive/ to move it into archive
  • Run ls archive and confirm new_name.txt is there
  • When ready, click the Test button

▶ 💡 Hints

  • The first argument to mv is the source, the second is the destination
  • After moving, the file should disappear from its original location
  • Use the Reset button to start over if needed
