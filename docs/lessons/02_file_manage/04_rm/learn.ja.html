削除とは

不要になったファイルやフォルダを消す操作を削除といいます。Linux では rm（remove）コマンドを使います。

Finder では、ファイルをゴミ箱に入れると一時的に退避され、空にするまで復元できることがあります。ターミナルの rm は通常、ゴミ箱を経由せず直接消える操作です。削除したデータは簡単には元に戻せないので、実行する前によく確認しましょう。

ファイルを削除する — rm

$ rm temp.txt

指定したファイルが消えます。ls で本当に消えたか確認する習慣をつけましょう。

ディレクトリを削除する — rm -r

フォルダを削除するときは -r オプションが必要です。フォルダの中身（子フォルダも含めて）ごとまとめて消します。

$ rm -r old_folder

空のフォルダだけ消す場合は rmdir という専用コマンドもあります（中身があると失敗します）。

$ rmdir empty_dir

削除の注意点

  • 本当に消していいか — ファイル名をよく読んでから Enter を押す
  • パスを間違えない — pwd で現在地を確認してから削除する
  • フォルダ削除は特に慎重に — rm -r は中身をすべて消します

本物の Linux では rm -rf / のような危険なコマンドでシステム全体を壊してしまう事故も報告されています。パスを必ず確認する習慣を身につけましょう。
