📝 ファイルとフォルダの削除

準備中...
English

課題

◀

📋 課題内容

temp.txt と junk.txt を削除し、空の empty_dir も削除しましょう。keep.txt は残してください。

✏️ やること

  • ls で削除対象と keep.txt を確認します
  • rm temp.txt で temp.txt を削除します
  • rm junk.txt で junk.txt を削除します
  • rmdir empty_dir で空のディレクトリを削除します
  • ls で keep.txt だけが残っていることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • 削除する前に pwd で現在地を確認しましょう
  • 空でないフォルダは rmdir では消せません（その場合は rm -r）
  • うまくいかないときは リセット ボタンで最初からやり直せます
