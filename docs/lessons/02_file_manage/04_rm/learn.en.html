What Is Deletion?

Deletion removes files or directories you no longer need. On Linux you use rm (remove).

Important: deleted data is hard to recover. Unlike a GUI trash bin, rm removes items immediately. Always double-check before pressing Enter.

Delete a File — rm

$ rm temp.txt

The specified file is removed. Use ls afterward to confirm.

Delete a Directory — rm -r

Directories require the -r (recursive) option to remove everything inside.

$ rm -r old_folder

For an empty folder only, you can use rmdir instead:

$ rmdir empty_dir

Safety Tips

  • Read the filename carefully before pressing Enter
  • Check your location with pwd before deleting
  • Be extra careful with directories — rm -r removes all contents

On real Linux systems, mistyped paths can cause serious damage. Build the habit of verifying paths first.
