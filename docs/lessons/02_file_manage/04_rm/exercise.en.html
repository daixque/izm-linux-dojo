📋 Task

Delete temp.txt and junk.txt, then remove the empty empty_dir folder. Leave keep.txt in place.

✏️ Instructions

  • Run ls to see what will be deleted and what should stay
  • Run rm temp.txt to delete temp.txt
  • Run rm junk.txt to delete junk.txt
  • Run rmdir empty_dir to remove the empty directory
  • Run ls and confirm only keep.txt remains
  • When ready, click the Test button

▶ 💡 Hints

  • Run pwd before deleting to confirm your location
  • Non-empty directories cannot be removed with rmdir (use rm -r instead)
  • Use the Reset button to start over if needed
