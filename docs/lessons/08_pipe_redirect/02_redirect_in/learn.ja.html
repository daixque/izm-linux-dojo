入力リダイレクト

📚 説明
English

ファイルから入力を受け取る

> が出力をファイルに送るのに対し、< はファイルの内容をコマンドの入力として渡します。

基本の形

コマンド < ファイル名

たとえば、次の 2 つは同じ意味です。

grep ERROR app.log
grep ERROR < app.log

2 つ目は app.log の内容が grep に渡されます（この例では grep にファイル名引数がない形）。

よく使う場面

パイプ（次のレッスン）と組み合わせる前に、< で「ファイルを入力源にする」イメージを押さえておきましょう。
