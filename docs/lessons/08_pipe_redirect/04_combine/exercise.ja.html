📋 課題内容

cat app.log | grep ERROR | wc -l > error_count.txt で ERROR 行の件数を数え、ファイルに保存しましょう。

✏️ やること

  • cat app.log | grep ERROR | wc -l > error_count.txt を実行します
  • cat error_count.txt で保存した件数（2）を確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • パイプ | で処理をつなぎ、最後に > でファイルに保存します
  • cat error_count.txt で 2 と表示されれば成功です
