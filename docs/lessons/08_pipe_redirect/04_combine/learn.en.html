Combine Search, Pipes, and Redirection

Combining what you learned in Chapters 7–8 enables practical workflows.

Typical Example: Save Results to a File

$ grep ERROR app.log > errors.txt

Only ERROR lines are written to errors.txt.

Pipe + Redirect

$ cat app.log | grep ERROR | wc -l > error_count.txt

Count ERROR lines and save that number to a file.

Workflow

  1. Read the data (cat or file argument)
  2. Filter to the lines you need (grep)
  3. Process it (wc, etc.)
  4. Save the result (>)
