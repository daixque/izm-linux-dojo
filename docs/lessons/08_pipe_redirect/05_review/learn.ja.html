第7–8章のまとめ

第7章ではファイルの探し方、第8章ではコマンド間のデータの流れを学びました。ここで振り返りましょう。

第7章 — 検索

コマンド・記法意味
grep パターン ファイルファイルの中身から、文字列を含む行を探す
grep -i/-n/-v/-c大文字小文字無視・行番号表示・反転・件数表示
find ディレクトリ -name パターンフォルダの下の階層すべてをたどり、ファイル名で探す
ls *.txtワイルドカードでファイル名を指定する

第8章 — パイプとリダイレクト

記号意味
>出力をファイルに書き込む（上書き）
>>出力をファイルに追記する
<ファイルの内容をコマンドの入力として渡す
|左のコマンドの出力を、右のコマンドの入力へ渡す

組み合わせの例

$ grep ERROR app.log > errors.txt
$ cat app.log | grep ERROR | wc -l

検索で必要なデータを取り出し、リダイレクトで保存したり、パイプで次の処理につなげたりできます。
