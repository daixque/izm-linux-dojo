第7–8章のまとめ

検索とデータの流れを制御するコマンドを振り返りましょう。

第7章 — 検索

コマンド意味
grep パターン ファイルテキストの中から行を検索
grep -i/-n/-v/-c大文字小文字無視・行番号・反転・件数
find dir -name パターンファイル名で再帰検索
ls *.txtワイルドカードでファイル名を指定

第8章 — パイプとリダイレクト

記号意味
>出力をファイルに書き込む（上書き）
>>出力をファイルに追記
<ファイルを入力として渡す
|コマンドの出力を次のコマンドへ
