📋 Task

Extract ERROR lines from a log and save the count to a file.

✏️ Instructions

  • Run cat app.log | grep ERROR | wc -l > error_count.txt
  • Run cat error_count.txt to verify the count (2)
  • Run grep ERROR app.log > errors.txt to save ERROR lines
  • Click the Test button when ready

▶ 💡 Hints

  • Pipes and redirection can be combined
  • Use cat to verify result files
