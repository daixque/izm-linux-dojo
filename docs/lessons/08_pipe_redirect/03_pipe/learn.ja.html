パイプ

📚 説明
English

コマンドをつなげる

パイプ（|）は、あるコマンドの出力を、次のコマンドの入力に渡す仕組みです。データがパイプ（管）のように流れるイメージです。

基本の形

コマンド1 | コマンド2

例: ログから ERROR を数える

$ cat app.log | grep ERROR | wc -l
2
  1. cat app.log — ファイル内容を出力
  2. grep ERROR — ERROR 行だけに絞る
  3. wc -l — 行数を数える

中間結果をファイルに保存しなくてよい

パイプを使えば、一時ファイルを作らずに処理をつなげられます。
