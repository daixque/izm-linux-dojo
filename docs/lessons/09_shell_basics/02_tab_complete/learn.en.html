Completing Commands and Paths with Tab

📚 Learn
日本語

Complete What You Type

Chapter 1 briefly introduced Tab completion. Here we cover it formally. Press Tab and the shell auto-completes command names and paths.

Complete Command Names

Type the first few characters of a command and press Tab. If there is only one match, it completes automatically.

$ gr[Tab]  →  grep  (when grep is the only match)

Complete Paths

Directory and file names can be completed with Tab too.

$ cd doc[Tab]  →  cd documents/

For directories, a trailing / may be added.

When There Are Multiple Matches

Pressing Tab twice may show a list of possible completions.
