第9章 レッスン2: コマンドとパスを Tab で補完する

📚 説明
English

入力を自動で補完する

コマンド名やファイル名は長いことがあり、全部を正確に打つのは大変です。Tab キーを押すと、シェルが入力途中の文字列を自動で補完してくれます。

第1章でも Tab を使ったことがあれば、同じ操作です。ここではコマンド名とパスの補完を整理して見ていきます。

コマンド名の補完

コマンドの最初の数文字を入力して Tab を押すと、候補が 1 つに決まれば自動で残りが補完されます。

$ gr[Tab]  →  grep

上の例では gr と入力したあと Tab を押すと、grep に補完されます（grep だけが候補の場合）。

パスの補完

ディレクトリ名やファイル名も Tab で補完できます。

$ cd doc[Tab]  →  cd documents/

ディレクトリの場合、末尾に / が付くことがあります。これは「フォルダの中に入る」という目印です。

候補が複数あるとき

Tab を 2 回押すと、入力に一致する候補の一覧が表示されることがあります。目的の名前を確認してから、もう少し文字を入力して Tab を押すと絞り込めます。
