コマンドの使い方を自分で調べる

📚 説明
English

困ったときは自分で調べる

Linux にはたくさんのコマンドがあり、すべてを暗記する必要はありません。使い方が分からなくなったら、ヘルプを調べる習慣をつけましょう。

コマンドには、自分の使い方を説明する機能が組み込まれています。2 つの代表的な方法を覚えておけば十分です。

--help / -h オプション

多くのコマンドは --help を付けると、短い使い方の説明を表示します。まずこちらを試すのが手軽です。

--help の短い形として -h があります。ターミナルでは -h の方がよく使われることもあります。多くのコマンドではどちらも同じヘルプを表示します。

ただし -h は、コマンドによっては別の意味を持つことがあります。例えば ls -h はヘルプではなく、ファイルサイズを読みやすい単位で表示するオプションです。ヘルプを見たいときは、まず --help を試すと確実です。

$ grep --help
Usage: grep [OPTION]... PATTERN [FILE]...
Search for PATTERN in each FILE.
...

オプションの一覧や簡単な説明が表示されます。画面が長くなることもありますが、スクロールして読めます。

man コマンド

man（manual の略）は、コマンドのマニュアルページを表示します。--help より詳しい説明が読めます。

$ man ls

man ページは専用の閲覧画面で開きます。上下キーやスペースキーでスクロールできます。

man を終了するときは q キーを押します。 ターミナルの通常の入力画面に戻ります。

使い分け

方法いつ使うか
コマンド --help（または コマンド -h）まず試す。短い説明で十分なとき
man コマンドもっと詳しく知りたいとき
