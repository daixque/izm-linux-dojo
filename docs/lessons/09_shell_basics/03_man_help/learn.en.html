Look It Up Yourself

Linux has many commands, and you do not need to memorize all of them. When you are unsure how something works, get in the habit of checking the help.

The --help / -h Option

Many commands show a brief description with --help. This is a quick place to start.

-h is the short form of --help. In the terminal, -h is used more often on some commands. On many commands, both show the same help text.

However, -h does not always mean help. For example, ls -h shows file sizes in a human-readable format—it is not the help page. When you want help, try --help first.

$ grep --help
Usage: grep [OPTION]... PATTERN [FILE]...
Search for PATTERN in each FILE.
...

The man Command

man (manual) displays a command's manual page. On real Linux you can read detailed documentation.

$ man ls

The dojo simulator also includes manual pages for major commands.

What to Remember

  • Try command --help first (or command -h on many commands)
  • Use man command when you want more detail
