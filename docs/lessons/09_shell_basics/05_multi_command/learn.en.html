Multiple Commands on One Line

The shell can run several commands in sequence on one line.

; — Run in Order

$ cd documents; ls

The right command runs regardless of whether the left one succeeded.

&& — Run If Previous Succeeded

$ cd documents && ls

The right command runs only if the left one succeeded (exit code 0). If cd fails, ls is not run.

When to Use Which

  • ; — run independent operations one after another
  • && — run the next step only if the previous one succeeded (safer chaining)
