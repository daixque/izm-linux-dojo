コマンド連結

📚 説明
English

1 行に複数コマンドを書く

シェルでは、1 行に複数のコマンドを書いて、順番に実行できます。コマンドとコマンドのあいだに区切りの記号を入れます。

区切りの記号によって、「前のコマンドが失敗したら次を実行するかどうか」が変わります。

; — 順番に実行

$ cd documents; ls

左のコマンドの成否に関係なく、右のコマンドも実行されます。cd が失敗しても ls は実行されます。

&& — 成功したら次を実行

$ cd documents && ls

左のコマンドが成功したときだけ、右のコマンドを実行します。cd が失敗したら ls は実行されません。

「成功」とは — 終了コード

コマンドが終わると、シェルには終了コードという数字が返されます。

  • 0 — 問題なく終了した（成功）
  • 0 以外 — 何か問題があった（失敗）

&& は、左のコマンドの終了コードが 0 のときだけ右を実行します。たとえば存在しないディレクトリへの cd は失敗（0 以外）になるので、その後の ls は動きません。

使い分け

記号使い方
;互いに関係ない操作を続けて実行するとき
&&前の操作が成功したときだけ次へ進みたいとき
