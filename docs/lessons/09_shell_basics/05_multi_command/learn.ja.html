コマンド連結

📚 説明
English

1 行に複数コマンド

シェルでは、1 行に複数のコマンドを書いて順番に実行できます。

; — 順番に実行

$ cd documents; ls

左のコマンドの成否に関係なく、右も実行されます。

&& — 成功したら次を実行

$ cd documents && ls

左のコマンドが成功（終了コード 0）したときだけ、右を実行します。cd が失敗したら ls は実行されません。

使い分け

  • ; — 独立した操作を続けて実行
  • && — 前が成功したときだけ次へ（安全な連鎖）
