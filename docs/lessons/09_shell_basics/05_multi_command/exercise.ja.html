📋 課題内容

cd documents && ls で documents に移動してから一覧表示しましょう。

✏️ やること

  • ls で documents があることを確認します
  • cd documents && ls を実行します
  • readme.txt が表示されることを確認します
  • 準備ができたら テスト ボタンをクリックしてください

▶ 💡 ヒント

  • && の前後にスペースを入れます
  • cd が失敗すると ls は実行されません
