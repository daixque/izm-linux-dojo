Inspecting Settings with Environment Variables

📚 Learn
日本語

Values the Shell Remembers

Environment variables are named values that the shell and programs can read. For example, the location of your home directory and your username are kept as environment variables.

Display with echo

echo prints a string. Reference environment variables with a $ prefix.

$ echo $HOME
/home/user

$ echo $USER
user

Common Variables

VariableMeaning
$HOMEHome directory path
$USERCurrent username
$PATHDirectories searched for commands

Why Know These?

Scripts and config files sometimes use $HOME. Environment variables let you confirm "where is my home?" without guessing.
