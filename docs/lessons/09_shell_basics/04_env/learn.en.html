Values the Shell Remembers

Environment variables are named values that the shell and programs can read.

Display with echo

$ echo $HOME
/home/user

$ echo $USER
user

Common Variables

VariableMeaning
$HOMEHome directory path
$USERCurrent username
$PATHDirectories searched for commands
