環境変数

📚 説明
English

シェルが覚えている値

環境変数は、シェルやプログラムが参照できる「名前付きの値」です。たとえば、ホームディレクトリの場所やユーザー名が環境変数として保持されています。

echo で表示

echo は文字列を表示するコマンドです。環境変数は $ を付けて参照します。

$ echo $HOME
/home/user

$ echo $USER
user

主な環境変数

変数意味
$HOMEホームディレクトリのパス
$USERログイン中のユーザー名
$PATHコマンドを探すディレクトリの一覧

なぜ知っておくか

スクリプトや設定ファイルで $HOME が使われることがあります。「自分のホームはどこか」を環境変数で確認できます。
