環境変数で設定値を確認する

📚 説明
English

シェルが覚えている値

シェルやプログラムは、作業に必要な情報を名前付きの値として保持しています。これを環境変数と呼びます。

たとえば「今ログインしているユーザー名は誰か」「ホームディレクトリはどこか」といった情報が、環境変数として保存されています。コマンドやスクリプトは、この値を参照して動作します。

echo で表示する

echo は文字列を表示するコマンドです。環境変数の値を見るには、変数名の前に $ を付けます。

$ echo $HOME
/home/user

$ echo $USER
user

$HOME にはホームディレクトリのパス、$USER にはユーザー名が入っています。

主な環境変数

変数意味
$HOMEホームディレクトリのパス
$USERログイン中のユーザー名
$PATHコマンドを探すディレクトリの一覧

$PATH には、ls や grep などのコマンドがどのフォルダから探されるかが書かれています。

なぜ知っておくか

設定ファイルやスクリプトで $HOME が使われることがあります。「自分のホームはどこか」を環境変数で確認できます。第1章で学んだ ~（ホームを表す記号）も、内部的には $HOME と同じ場所を指します。
