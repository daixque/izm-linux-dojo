Recall Commands You Typed

When you started using the terminal in Chapter 1, you may have noticed that the ↑ key brings back previous commands. Here we formally cover command history.

What Is History?

The shell records commands you run after pressing Enter as history. You do not need to retype long commands every time.

↑ and ↓ Keys

  • ↑ — show the previous command (press again for older ones)
  • ↓ — move toward newer commands (use after going back with ↑)

Tips

  • When running the same command repeatedly, press ↑ and Enter
  • When you need a small change, press ↑, edit, then Enter
